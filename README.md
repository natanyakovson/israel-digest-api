# Israel Digest News API

Веб-сервис для Make.com. Собирает свежие сообщения из заданных публичных Telegram-каналов, выполняет базовое удаление дублей и возвращает JSON.

## Адреса

- `/health` — проверка работы сервиса
- `/news?hours=12` — новости за последние 12 часов
- `/news?since=2026-07-19T07:00:00+03:00` — новости после конкретного времени

## Параметры `/news`

- `hours`: от 1 до 48, по умолчанию 12
- `since`: ISO-время; если указано, имеет приоритет над `hours`
- `limit`: максимум возвращаемых новостей, по умолчанию 100
- `dedup_threshold`: чувствительность удаления дублей, по умолчанию 0.55

## Авторизация

Render автоматически создаёт переменную `DIGEST_API_KEY`. Передавайте её в заголовке:

```text
Authorization: Bearer ВАШ_КЛЮЧ
```

## Настройка модуля HTTP в Make

- Method: `GET`
- URL: `https://ИМЯ-СЕРВИСА.onrender.com/news?hours=12`
- Header name: `Authorization`
- Header value: `Bearer ВАШ_КЛЮЧ`
- Parse response: `Yes`

Массив новостей находится в поле `posts`.
